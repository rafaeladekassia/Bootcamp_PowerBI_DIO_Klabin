# Hands-on Project: Ecommerce Database
## Project Description

The project aims to create from scratch a database for an Ecommerce platform to manage **clients, orders, products, sellers, and product inventory**, as well as data analysis for businesss insights.  The database was created according to the following steps:

* writing the SQL script for al the tables included;
* inserting data;
* creating queries to solve specific business questions.

> *All data used here is fictional and for educational purpose.*

## Business Questions

The following business questions were considered in this project:
* How many orders did each client place?
* What products did customers buy?

## Data modelling
The script makes use of the following main entities:

* **Clients:** Register clients through an unique identifier (Tax ID).
* **Product:** Register products available in certain categories.
* **Orders:** Store information on orders placed by clients.
* **Product inventory:** Manage product location and availability.
* **Supplier:** Identify product suppliers and related information.
* **Seller:** Register individual sellers on the database.
